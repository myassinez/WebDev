CSS Syntax

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of HTML documents.

The basic CSS syntax consists of selectors and declaration blocks. A selector points to the HTML element you want to style.

Each declaration consists of a property and a value. Multiple declarations are separated by semicolons.

selector { property: value; } h1 { color: blue; } p { font-size: 16px; }
  • CSS rules start with a selector
  • Properties define the style characteristics
CSS Selectors

CSS selectors are patterns used to select the HTML elements you want to style.

They allow you to target specific elements based on various criteria.

Selectors can be combined to create more specific targeting.

element { } .class { } #id { }
  • Element Selector: Targets HTML elements by tag name
  • Class Selector: Targets elements with specific class
  • ID Selector: Targets single element with specific ID
CSS Properties

CSS properties define the visual characteristics of HTML elements.

Each property has a specific purpose in styling elements.

Properties work together to create the overall appearance.

color: red; font-size: 16px; margin: 20px;
  • Color properties control colors
  • Text properties control typography
  • Layout properties control positioning
CSS Box Model

The CSS box model describes the rectangular boxes generated for elements.

Each box has content, padding, borders, and margins.

Understanding the box model is crucial for layout design.

margin: 10px; padding: 20px; border: 1px solid black;
  • Content: The actual content area
  • Padding: Space inside the border
  • Border: Line surrounding padding and content
CSS Flexbox

Flexbox is a one-dimensional layout method for arranging items in rows or columns.

It provides efficient space distribution and alignment within a container.

Flexbox simplifies many common layout challenges.

display: flex; flex-direction: row; justify-content: center;
  • Container becomes a flex container
  • Items become flex items
  • Main axis and cross axis define direction
CSS Grid

CSS Grid Layout is a two-dimensional layout system for the web.

It allows creation of complex layouts with rows and columns.

Grid makes responsive design more manageable.

display: grid; grid-template-columns: 1fr 1fr; grid-gap: 10px;
  • Define grid container
  • Set up rows and columns
  • Place items in grid areas
CSS Colors

CSS provides several ways to specify colors for elements.

Colors can be defined using various notation systems.

Understanding color values is important for consistent design.

color: red; background-color: #FF5733; border-color: rgb(255, 0, 0);
  • Named colors are predefined
  • Hexadecimal uses hex values
  • RGB uses red, green, blue values
CSS Typography

CSS provides extensive control over text appearance and layout.

Typography properties adjust font characteristics and spacing.

Good typography enhances readability and user experience.

font-family: Arial, sans-serif; font-size: 1rem; line-height: 1.5;
  • Choose appropriate font families
  • Set readable font sizes
  • Adjust line spacing for readability
CSS Responsive Design

Responsive design ensures web pages look good on all devices.

It adapts layout to different screen sizes and orientations.

Media queries are key to responsive design implementation.

@media (max-width: 768px) { } width: 100%; max-width: 1200px;
  • Use media queries for breakpoints
  • Employ flexible units
  • Test on multiple devices
CSS Transitions and Animations

CSS allows creation of smooth transitions between property values.

Animations offer more control over timing and sequence.

These features enhance user interaction and engagement.

transition: all 0.3s ease; animation: slide 2s infinite; @keyframes slide { }
  • Transitions for simple state changes
  • Animations for complex sequences
  • Keyframes define animation steps
